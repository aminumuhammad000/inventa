Import Inventory from CSV

CSV Format Requirements:

Your CSV file should have the following columns in this order:

  • Name - Item name (required)
  • SKU - Stock keeping unit (required)
  • Category - Item category (required)
  • Current Stock - Current quantity (required)
  • Unit - Unit of measurement (required)
  • Cost Price - Purchase price (required)
  • Selling Price - Sale price (required)
  • Min Stock Level - Minimum stock alert (required)

Note: The first row should contain headers. Existing items with the same SKU will be updated.
